# Cognitive Reference Charts

This repository hosts an interactive website for exploring  cognitive reference charts using both simulated as well as publicy available datasets. The methods and results behind these charts are described in the accompanying research papers, which will hopefully be published soon 🤞.

The website is available at:
https://bbuckova.github.io/Cognitive_Growth_Charts/

Here users can explore different data issues, experiment with different modeling settings and examine how the interactive effects between these two can be modeled through cognitive reference charts.
