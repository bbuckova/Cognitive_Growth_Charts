About This Dashboard

This interactive dashboard presents normative models for various cognitive measures. Our research has developed normative models that account for demographic variables and provide standardized reference points for cognitive assessment.

How to Use

Select a cognitive measure from the left sidebar to view interactive visualizations. Each measure displays four coordinated charts showing different aspects of the normative model. Hover over data points to see detailed information and observe how the same subjects are highlighted across all charts.

Key Features

Cross-Chart Highlighting: Hover over any data point to see the same subject highlighted across all four visualizations simultaneously.

Export Functionality: Download data, model parameters, and high-resolution images for publications.

Responsive Design: Works seamlessly on desktop, tablet, and mobile devices.

Real-time Analysis: Explore normative data interactively without needing specialized software.

Research Background

These normative models were developed as part of ongoing research into cognitive assessment standardization. The models account for demographic factors and provide clinically relevant reference points for cognitive performance evaluation.

Getting Started

Click on any cognitive measure in the sidebar to begin exploring the normative data. Each visualization provides a different perspective on the relationship between demographic variables and cognitive performance.
