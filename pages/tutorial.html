How to use the cognitive reference charts?

The website has been built to allow the utilization of the cognitive reference charts by anyone as a reference model for their own data. Reference charts are particularly beneficial when the sample size of your data is too small to create your own model. The normative models for each cognitive variable can be downloaded and used to transfer or extend the model with your data. In this tutorial, we will focus on the transfer framework, you can find more information in the pcntoolkit tutorial. The transfer approach allows for the creation of new local norms. This method derives a new local normative model on the new dataset from the trained normative model without refitting the parameters of the existing sites.

Step by step:

  1. Export the model

    2. The trained normative model can be exported at the top of each page by clicking on “Export Model”. This will download a folder that contains the model parameters.

  2. Install the PCN toolkit

    3. All the information to install and set up the pcntoolkit package is available on the online tutorial.

  3. Load the trained model from the cognitive reference chart

    4. Load the trained normative model in your environment and extract the exact names for each variable. 

                
                fitted_model = NormativeModel.load("/Downloads/NIH_Flanker_RawScore_model")
                
                # Extract model configuration: batch effects (e.g. site, sex), covariates (e.g. age), and response variables (brain measures predicted by the model)
                fitted_batch_effects = list(fitted_model.unique_batch_effects.keys())
                fitted_covariates = fitted_model.covariates
                fitted_response_vars = fitted_model.response_vars
  4. Configure the HBR model

    5. Here, you need to use the same model that was used to fit the trained model. You can check the scripts used on the github. 

                
                #Configure the hbr model
                mu = make_prior(
                linear=True,
                slope=make_prior(dist_name="Normal", dist_params=(0.0, 10.0)),
                intercept=make_prior(
                    random=True,
                    mu=make_prior(dist_name="Normal", dist_params=(0.0, 1.0)),
                    sigma=make_prior(dist_name="Normal", dist_params=(0.0, 1.0), mapping="softplus", mapping_params=(0.0, 3.0)),
                ),
                basis_function=BsplineBasisFunction(basis_column=0, nknots=5, degree=3),
                )
                
                sigma = make_prior(
                linear=True,
                slope=make_prior(dist_name="Normal", dist_params=(0.0, 2.0)),
                intercept=make_prior(dist_name="Normal", dist_params=(1.0, 1.0)),
                basis_function=BsplineBasisFunction(basis_column=0, nknots=5, degree=3),
                mapping="softplus",
                mapping_params=(0.0, 3.0),
                )

                likelihood = NormalLikelihood(mu, sigma)

                template_hbr = HBR(
                    name="template",
                    cores=16,
                    progressbar=False,
                    draws=1500,
                    tune=500,
                    chains=4,
                    nuts_sampler="nutpie",
                    likelihood=likelihood,
                )
  5. Load your dataset

    6. In order to extend or transfer the reference charts to your data, your data should be in the same format, with the same names for the variables and the same scale. 

                
                #Prepare the new dateset
                #load csv file
                new_data = pd.read_csv('/project//new_dataset.csv', sep=';')
                new_data.columns
                
                #Rename the site column or add site as a new column if your dataset does not contain different sites
                new_data["Site"] = "site_new_data"
                
                #Change the name of covariates, batch effects and response variable to align with the trained model
                new_data= new_data.rename(columns={'CogVariable': fitted_response_vars[0]})
                
                # Recode sex variable to match the format expected by the model (F/M instead of 0/1) 
                new_data["Sex"] = new_data["Sex"].map({"F":0.0,"M":1.0})
                
                #Check the scale, for instance here the age for the reference charts is in months
                new_data["Age"]=new_data["Age"]/12
  6. Create a NormData object

    7. Make the new dataset into a NormData object (see the pcntoolkit tutorial for more information). 

                
                # specify the column names to use
                covariates = ["Age"]
                batch_effects = ["Sex","Site"]
                response_vars = ["NIH_Flanker_RawScore"]
                
                #Make the new_data into a NormData object
                norm_new_data = NormData.from_dataframe(
                    name="New Dataset",
                    dataframe=new_data,
                    covariates=covariates,
                    batch_effects=batch_effects,
                    response_vars=response_vars,
                    remove_outliers=True,
                    z_threshold=10,
                )
                
                norm_new_data.coords
  7. Split the dataset into a train and a test samples

    8. The decision on who to put in the train sample is important as you will not obtain the Z-scores for those individuals. Here, we propose two options: 

                
                #Split the new dataset - basic function if your sample is homogenous
                train_norm_new_data, test_norm_new_data = (norm_new_data.train_test_split())
                
                #Split the new dataset by hand if, for instance, you want the controls in the test dataset and the patients in the train dataset
                #separate the patients (1) from the controls(0) before making the normdata object
                patient_new_data = new_data[(new_data["Patient"] == 1)]
                test_new_data = new_data[(new_data["Patient"] == 0)]
                    
                #make subsets into normdata objects
                train_norm_new_data = NormData.from_dataframe(
                    name="New Dataset",
                    dataframe=test_new_data,
                    covariates=covariates,
                    batch_effects=batch_effects,
                    response_vars=response_vars,
                    remove_outliers=True,
                    z_threshold=10,
                )
                test_norm_new_data = NormData.from_dataframe(
                    name="New Dataset",
                    dataframe=patient_new_data,
                    covariates=covariates,
                    batch_effects=batch_effects,
                    response_vars=response_vars,
                    remove_outliers=True,
                    z_threshold=10,
                )
  8. Transfer the data

    9. You can now transfer the cognitive reference charts to your NormData dataset and visualize the results. 

                
                #transfer
                transferred_with_new_data = fitted_model.transfer_predict(
                    train_norm_new_data,
                    test_norm_new_data,
                    save_dir=(
                        "/home/Downloads/transfer/"
                    ),
                );
                
                plot_centiles_advanced(
                    transferred_with_new_data,
                    scatter_data=train_norm_new_data,
                    covariate_range=(5,100),
                    batch_effects="all",
                )
  9. Get the Z-scores

    10. Finally, you can extract your Z-scores from the train_norm_new_data object. 

                
                train_norm_new = train_norm_new_data.to_dataframe()
                train_norm_new['Z']
