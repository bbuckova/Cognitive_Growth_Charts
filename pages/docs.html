Methodology

The normative models were implemented using the HBR framework from PCNtoolkit (v1.1.2). Cognitive measures were modeled as nonlinear functions of age using B-spline basis functions, while sex and site were included as batch-effect covariates.

• Hierarchical Bayesian Regression (HBR): Hierarchical normative modeling with centile estimation

• B-spline Modeling: Flexible nonlinear modeling of age-related cognitive trajectories

• Batch Effect Correction: Site and sex modeled as random intercept effects

• Bayesian Sampling: Models fitted using PyMC with NUTS sampling (2 chains, 1500 posterior samples)

• Model Evaluation: Performance assessed using train/test splits, Q-Q plots, R², RMSE, MAE, skewness, and kurtosis metrics

Data Sources

The normative reference charts were developed using a large aggregated cognitive dataset spanning the human lifespan. The current dataset (7th of May 2026) includes 14,421 healthy participants aged 5.0 to 100 years collected across multiple large-scale research cohorts.

Included datasets:

• HCP1200: N = 1,206, ages 22–37 years, 54% female

• HCP Aging: N = 720, ages 36–100 years, 56% female

• HCP Development: N = 641, ages 5.0–21.9 years, 54% female

• ABCD Study: N = 11,854, ages 8–11 years, 48% female

All data were anonymized and harmonized across sites prior to normative modeling to preserve participant privacy.

Chart Descriptions

Raw Score vs Age: Displays observed cognitive scores across the lifespan for each dataset before harmonization.

Harmonized Score vs Age with Centiles: Shows normative trajectories after HBR harmonization, including estimated centile curves across age.

Q-Q Plot: Evaluates how closely the standardized scores follow a normal distribution across datasets.

Z-Score Distribution: Displays the distribution of HBR harmonized deviation scores for each dataset.
